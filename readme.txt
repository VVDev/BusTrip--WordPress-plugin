BusTrip- Wordpress plugin per il portale web di un azienda di servizi per la mobilità.

Visualizzazione delle news;
Cerca per linea;
Cerca per fermata;
Programma viaggio;
Calcola biglietto.


############# INSTALLATION GUIDE ################

Dopo aver installato e attivato il plugin,
1)Usare lo shortcode [bustrip] nella pagina dove si vuole visualizzare la web app.
